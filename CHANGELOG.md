## [0.1.0] - 2025-11-05

### Добавлено

#### Этап 1: Окружение разработки
- Настроена среда разработки с Node.js 24.11.0
- Создан проект на Vite + React + TypeScript
- Установлены зависимости: react-router-dom, dexie, vite-plugin-pwa

#### Этап 2: Структура проекта
- Создана полная структура папок проекта
- Добавлен PWA manifest.json с настройками приложения
- Создан базовый Service Worker для офлайн-работы
- Определены TypeScript типы для всех сущностей:
  - Card (карточка изображения)
  - Tag (метка)
  - Category (категория меток)
  - Collection (коллекция)
  - Moodboard (мудборд)
  - AppSettings (настройки)
  - SearchHistory (история поиска)
  - ViewHistory (история просмотров)
  - ThumbnailCache (кеш превью)
  - AppStatistics (статистика)

#### Этап 3: Дизайн-система
- Реализована система CSS переменных с полной цветовой палитрой:
  - Grayscale (25-900) — основная шкала
  - Red (100-900) — для ошибок
  - Yellow (100-900) — для предупреждений
  - Green (100-900) — для успешных действий
- Настроена типографика с двумя шрифтами:
  - YS Music Headline Bold — для заголовков
  - Geologica Variable — для текста
- Создана система отступов, скруглений, теней
- Добавлены глобальные стили и утилиты

#### Этап 4: Базовые UI-компоненты
- **Button** — универсальная кнопка:
  - 5 вариантов: primary, secondary, ghost, danger, success
  - 3 размера: small, medium, large
  - Состояния: default, hover, active, disabled, loading
  - Поддержка иконок и fullWidth режима
- **Tag** — метка для категоризации:
  - 3 варианта: default, active, overlay
  - Возможность удаления
  - Счётчики количества
  - Поддержка кастомных цветов
- **Input** — поле ввода:
  - Поддержка иконок слева/справа
  - Состояния: error, disabled, focus
  - Кнопка очистки (clearable)
  - Метки и сообщения об ошибках
- **Card** — карточка изображения:
  - Превью с lazy loading
  - Checkbox для выбора
  - Overlay с информацией
  - Бейджи типа файла и мудборда
  - Компактный режим отображения

#### Этап 5: Навигация и Layout
- **Sidebar** — боковое меню:
  - Иконки разделов с tooltips
  - Активное состояние текущей страницы
  - Адаптивность под высоту окна
- **SearchBar** — поисковое меню:
  - Поле поиска с саджестом
  - Отображение выбранных меток
  - Выпадающее меню (базовая версия)
- **SectionHeader** — меню разделов:
  - Заголовок текущего раздела
  - Переключатель вида (стандартный/компактный)
  - Фильтры контента (всё/изображения/видео)
  - Счётчики элементов
  - Дополнительные кнопки действий
- **Layout** — основной лэйаут:
  - Интеграция всех навигационных компонентов
  - Фиксированный Sidebar и Header
  - Скроллируемый контент
- **Страницы приложения**:
  - CardsPage — главная страница с галереей
  - CollectionsPage — страница коллекций
  - TagsPage — управление метками и категориями
  - MoodboardPage — мудборд
  - SettingsPage — настройки
  - AddPage — добавление карточек
- **React Router** — настроена навигация между страницами

#### Этап 6: IndexedDB
- Создан сервис для работы с базой данных на Dexie.js
- Определена схема БД с индексами для оптимизации
- Реализованы CRUD операции для всех сущностей:
  - Cards: add, get, getAll, update, delete, search
  - Tags: add, getAll, update, delete
  - Categories: add, getAll, update, delete
  - Collections: add, get, getAll, update, delete
  - Moodboard: get, addCard, removeCard, clear
- Добавлены функции для:
  - Поиска и фильтрации карточек
  - Получения статистики
  - Работы с историей поиска и просмотров
  - Управления кешем превью
