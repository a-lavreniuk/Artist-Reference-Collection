# Установка зависимостей

```bash
npm install
```

или

```bash
yarn install
```

или

```bash
pnpm install
```

# Разработка

Запуск dev-сервера:

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

# Сборка

Создание production build:

```bash
npm run build
```

Результат будет в папке `dist/`

# Предпросмотр production build

```bash
npm run preview
```

# Проверка типов

```bash
npm run type-check
```

# Линтинг

```bash
npm run lint
```
