## Для какой сборки актуален список
Ниже — известные ограничения и обходные пути для сборки 0.1.4. Если у вас другая версия или поведение не совпадает с описанием, оформите отчёт по статье «Сообщение о багах».

## SmartScreen при установке
Симптом: Windows блокирует установщик, пишет про неизвестного издателя.
**Причина:** у установщика пока нет коммерческой подписи кода.
**Обход:** нажмите **«Подробнее»** → **«Выполнить в любом случае»**, скачав файл только с GitHub Release.

## Одна библиотека на установку
Симптом: нельзя подключить вторую независимую папку-библиотеку параллельно с первой.
**Обход:** переключатель в навбаре меняет **область просмотра** (вся библиотека / без меток / корзина), а не другую библиотеку на диске.

## Повторный онбординг из меню
Симптом: в «Меню» нет пункта, который снова показывает приветственный экран.
**Обход:** тур интерфейса — **«Настройки → Общие → Знакомство с интерфейсом»**.

## Горячие клавиши только для просмотра
Симптом: в разделе «Горячие клавиши» нельзя переназначить сочетания — список только для справки.

## UI-Kit в меню
Симптом: пункт «UI-Kit» в «Меню» открывает служебную демо-страницу дизайн-системы.
**Обход:** при пользовательском тестировании этот пункт можно игнорировать.

## macOS — тестовые сборки
Симптом: DMG для macOS пока не публикуется в Releases автоматически — только через GitHub Actions.
