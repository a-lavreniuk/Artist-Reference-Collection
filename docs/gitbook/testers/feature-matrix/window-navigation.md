## Окно и навигация
### Top bar
- [ ] Назад / Вперёд — история маршрутов
- [ ] Заголовок окна меняется по разделу
- [ ] Свернуть в трей
- [ ] Развернуть / восстановить окно
- [ ] Закрыть — сворачивание в трей (если включено в настройках)
### Вкладки
- [ ] Библиотека, Коллекции, Мудборд переключаются
- [ ] Доска — заглушка «в разработке»
- [ ] При уходе с Библиотеки scope сбрасывается на «Вся библиотека»
### Переключатель библиотеки
- [ ] Вся библиотека / Без меток / Корзина
- [ ] Корзина: «Очистить» с подтверждением
### Меню
- [ ] Категории и метки, Статистика, История, Настройки
- [ ] Поиск дублей — скан, сравнение, слияние
- [ ] Размер сетки L / M / S
- [ ] Тема: светлая / тёмная / автоматическая
- [ ] Сообщить о проблеме
### Прочее
- [ ] Кнопка «Добавить» — выбор файлов
- [ ] Maintenance overlay при бэкапе/миграции
- [ ] Scroll to top на длинных сетках
