## Поиск, фильтры, сортировка
### Поиск — 4 режима
- [ ] Метки: чипы, ID карточки, недавние
- [ ] AI Семантика: запрос, выдача после индексации
- [ ] По цвету: HEX, пресеты, точность
- [ ] Похожие: файл / недавние / с карточки
- [ ] Раскрытие панели, сброс, сохранение режима
### Фильтры (8 категорий)
- [ ] Соотношение сторон, тип файла, описание, ссылка
- [ ] Дата добавления, вес, разрешение, длительность
- [ ] Настроить список, пресеты, очистить
### Сортировка
- [ ] Дата, тип, вес, разрешение, длительность + направление
- [ ] Перемешать (только Библиотека)
- [ ] Сохранение после перезапуска
