## Назначение
Единый реестр **всего** пользовательского функционала ARC. Отмечайте пункты при каждой сборке — так ни одна фича не пропадёт из проверки.
**Версия:** 0.1.4 · **Дата:** 2026-07-04
 ---
## Как пользововаться
1. Сначала — **Сценарные прогоны** (сквозные пути).
2. Затем — все 7 областей ниже (полное покрытие).
3. Детальные шаги — в **Чек-листах тестирования**.
