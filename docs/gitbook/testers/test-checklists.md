## Как пользоваться чек-листами
## Отмечайте пункты по мере проверки сборки 0.1.4. Если что-то ломается — запишите шаги и загляните в Console по инструкции из «DevTools и логи».
## Установка и запуск
- [ ] Установщик с GitHub проходит SmartScreen и завершается без ошибок.
- [ ] При первом запуске выбирается папка библиотеки, главное окно открывается на **«Библиотеке»**.
- [ ] Иконка в трее разворачивает окно из свёрнутого состояния.
- [ ] При закрытии крестиком с настройкой «сворачивать в трей» процесс ARC остаётся в диспетчере задач.
## Импорт и библиотека
- [ ] Drag-and-drop 3–5 изображений — карточки появляются с превью.
- [ ] **«Добавить»** с множественным выбором даёт тот же результат.
- [ ] При импорте файла, который уже есть в библиотеке, открывается модалка разрешения конфликта.
- [ ] Сортировка по дате меняет порядок карточек.
- [ ] **«Фильтры → Тип файла»** «только изображения» скрывает видео.
- [ ] Сохранённый пресет фильтров переживает перезапуск.
- [ ] При активном поиске полоса коллекций скрыта; после сброса — снова видна.
- [ ] **«Без меток»** и **«Корзина»** показывают корректный набор карточек.
- [ ] Мягкое удаление и восстановление из корзины работают.
## Карточка и метки
- [ ] Клик по карточке открывает оверлей с медиа.
- [ ] Две метки в оверлее находятся через поиск.
- [ ] **«Найти похожее»** без AI ищет по меткам.
- [ ] Поиск по метке фильтрует сетку.
## Коллекции и мудборд
- [ ] **«Коллекции»** — sidebar и сетка отображаются корректно.
- [ ] Полоса коллекций видна на **«Библиотеке»**.
- [ ] Порядок полосы настраивается в **«Настройки → Общие»**.
- [ ] Создание коллекции и добавление карточки работают.
- [ ] Переименование и drag-reorder коллекций сохраняются.
- [ ] На мудборде карточка появляется на вкладке **«Мудборд»**.
## Расширение браузера
- [ ] В **«Настройки → Расширение браузера»** включается Import API и показывается статус подключения.
- [ ] Расширение в Chrome/Edge импортирует изображение с открытой страницы в библиотеку ARC.
- [ ] Повторный импорт того же изображения не создаёт вторую карточку.
## Поиск дублей
- [ ] Пункт **«Меню» → «»** открывает раздел с группами похожих карточек.
- [ ] Сравнение двух карточек в группе показывает превью и доступные действия.
- [ ] Удаление или объединение дублей обновляет сетку без перезапуска.
## Инструменты
- [ ] Скриншот **Ctrl+Alt+E** создаёт карточку в библиотеке.
- [ ] **Сообщить о проблеме** и **Ctrl+Shift+B** открывают форму в браузере.
- [ ] **«История»** и **«Статистика»** обновляются после импорта.
- [ ] **«Оформление»** меняет тему интерфейса.
## Настройки и данные
- [ ] Резервная копия создаёт файл на диске.
- [ ] Проверка целостности на здоровой библиотеке не находит ошибок.
- [ ] При «осиротевших» записях и лишних файлах на диске отчёт показывает понятный список.
- [ ] Мастер исправления предлагает действия и завершается без потери данных.
- [ ] Автоимпорт из наблюдаемой папки подхватывает новые файлы.
## AI и обновления
- [ ] Модель загружается, индексация завершается.
- [ ] **«AI Семантика»** даёт выдачу после индексации.
- [ ] **«Похожие»** и **«По цвету»** работают ожидаемо.
- [ ] **«Проверить обновления»** находит или корректно сообщает об отсутствии версии.
- [ ] После обновления показывается **«Что нового?»**.
