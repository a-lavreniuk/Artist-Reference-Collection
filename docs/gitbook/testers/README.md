Этот раздел собран для бета-тестеров ARC. Здесь — всё, что нужно, чтобы поставить приложение, понять, где лежат ваши данные, настроить AI-поиск, при необходимости заглянуть в DevTools, свериться с известными ограничениями и оформить понятный отчёт о проблеме.

Тон здесь чуть техничнее, чем в пользовательских статьях: мы не уходим во внутренности кода, но предполагаем, что вы готовы внимательно прочитать путь на диске или скопировать текст ошибки из консоли.
