При первом запуске ARC показывает **онбординг** — короткую цепочку экранов, которая знакомит с продуктом и помогает выбрать папку библиотеки на диске. Это не длинный курс: несколько шагов, после которых открывается главное окно, и при желании запускается **тур по интерфейсу** с подсветкой основных зон.
## Что в онбординге
Экран **«Добро пожаловать в ARC»** кратко объясняет, зачем нужно приложение.
**«Преимущества ARC»** перечисляет ключевые возможности; там же есть ссылка на эту базу знаний. На шаге **подключения библиотеки** вы указываете или создаёте папку, куда ARC будет складывать импортированные файлы, превью и служебные данные. Без этого шага приложение не знает, где хранить вашу коллекцию.
Тур по интерфейсу можно пройти повторно: **«Настройки → Общие»** → **«Знакомство с интерфейсом»**.
## Первый импорт
Когда библиотека подключена и окно открыто, добавьте первые референсы. Основной способ — кнопка **«Добавить»** в навбаре и выбор файлов в системном диалоге. Второй — **перетащить** изображения или видео прямо в окно **«Библиотеки»**: появится drop зона, после отпускания файлы импортируются в сетку.
Карточки появятся с превью сразу после импорта. Метки и коллекции можно наращивать постепенно — для первого знакомства достаточно открыть одну карточку в оверлее и посмотреть, как устроен просмотр.
