Прежде чем углубляться в библиотеку, поиск и фильтры, полезно один раз спокойно разобраться, из чего состоит окно ARC. В этом разделе — обзорные статьи по зонам интерфейса: верхняя полоса, навбар, вкладки, меню **«⋯»**, знакомство с поиском и первый импорт. Каждая статья описывает одну область экрана, без детального разбора каждой кнопки — это уже в **«3. Руководство по приложению»**.

Если открываете приложение впервые, начните с **«Обзор окна ARC»** — там общая схема. Затем можно пройти остальные статьи по порядку или вернуться к нужной, когда столкнётесь с конкретным элементом на экране.
