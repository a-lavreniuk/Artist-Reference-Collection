Раздел **«Настройки»** собирает всё, что влияет на поведение ARC за пределами конкретной карточки или коллекции: внешний вид, папку библиотеки на диске, резервные копии, уведомления, горячие клавиши, автоимпорт, расширение браузера, AI-поиск и обновления приложения. Открыть настройки можно из **Меню** или перейти на экран настроек из навбара — слева появится sidebar с группами панелей, справа содержимое выбранного раздела.
