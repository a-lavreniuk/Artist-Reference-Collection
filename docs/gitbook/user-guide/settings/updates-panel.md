**«Настройки → Обновления»** — проверка **новых версий ARC**, установка скачанного обновления и просмотр **«Что нового?»** после обновления. Обычно здесь видна **текущая версия** — её стоит указывать в отчётах о проблемах. При доступной новой сборке приложение может предложить установку сразу или через уведомление — в зависимости от настроек.
Перед обновлением на крупный релиз при большой библиотеке разумно сделать **резервную копию**. После обновления загляните в release notes: иногда меняется поведение импорта, индексации или формата библиотеки.
