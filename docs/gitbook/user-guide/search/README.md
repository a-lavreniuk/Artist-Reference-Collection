Поиск в ARC — это не одна строка «найти по названию», а целый набор способов отобрать референсы под задачу. В центре навбара живёт островок с четырьмя режимами: по **меткам**, по смыслу через **AI Семантику**, по **доминирующему цвету** и по **визуальному сходству**. Отдельно — быстрый вход в карточку по её **ID**, если вы уже знаете, что ищете. Режимы переключаются иконками слева от поля; после клика панель разворачивается и показывает подсказки, недавние метки и просмотренные карточки.
Поиск работает вместе с **«Фильтрами»** — иконкой справа в навбаре. Можно сначала сузить сетку запросом, а потом уточнить техническими условиями, или наоборот. На вкладках **«Библиотека»**, **«Коллекции»** и **«Мудборд»** поиск всегда действует на ту сетку, которую вы видите сейчас — всю библиотеку, содержимое коллекции или карточки мудборда.
Ниже — обзор общей логики и отдельные статьи по каждому режиму. Если вы только знакомитесь с ARC, начните с **«Обзор поиска в навбаре»**; дальше переходите к тому режиму, который ближе к вашей задаче.
