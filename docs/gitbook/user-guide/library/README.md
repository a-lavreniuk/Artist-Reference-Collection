**«Библиотека»** — сердце ARC и экран, с которого начинается почти каждая сессия. Здесь лежат все импортированные референсы: сотни превью в одной сетке, без необходимости открывать каждый файл отдельно. Именно сюда попадают карточки после импорта, сюда вы возвращаетесь, когда нужно увидеть базу целиком или разобрать свежую пачку файлов без меток.
В отличие от **«Коллекций»** и **«Мудборда»**, библиотека показывает не подборку, а полный архив (за исключением карточек в корзине). Здесь работают поиск и **«Фильтры»** по всей базе, переключатель областей (**«Вся библиотека»**, **«Без меток»**, **«Корзина»**), сортировка и кнопка **«Добавить»**. Над сеткой может появиться полоса коллекций — быстрый переход в подборку, не меняя вкладку.
Статьи ниже разбирают библиотеку по частям: как устроена сетка, как импортировать файлы, чем отличаются области просмотра, что делать при пустой базе или пустой выдаче по фильтрам, как настроить порядок и размер карточек. Начните с обзора, если вы в ARC впервые; если уже работаете в сетке — переходите к нужной теме.
