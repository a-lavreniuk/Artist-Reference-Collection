Сетка **«Библиотеки»** не застывает в одном порядке. **Сортировка** меняет последовательность карточек — по дате добавления, размеру файла, разрешению, длительности видео и другим полям. **Размер сетки** меняет плотность превью — сколько карточек помещается на экране без изменения самих файлов.
Оба инструмента дополняют поиск и **«Фильтры»**: сначала отбираете подмножество (например, только видео), затем сортируете по длительности или показываете в компактной сетке для обзора.
На **«Библиотеке»** есть особый пункт **«Перемешать»** — случайный порядок, который сохраняется, пока вы снова не выберете другую сортировку. В **«Коллекциях»** и на **«Мудборде»** сортировка тоже есть, но без перемешивания всей базы.
## Где найти
Кнопка **сортировки** — в навбаре **слева от иконки «Фильтры»**. Нажмите на неё: откроется меню **«Сортировка»** со списком полей и направлением — **«Сначала новые»**, **«Сначала старые»**, по возрастанию или убыванию для числовых параметров.
Размер сетки **не** вынесен в навбар. Откройте меню **Меню** в шапке и выберите **«Размер сетки»** — **Большая**, **Средняя** или **Маленькая**. Настройка действует на текущий экран с сеткой (**«Библиотека»**, **«Коллекции»**, **«Мудборд»**).
Чтобы показать только изображения или только видео, используйте **«Фильтры»** и пункт про тип файла.
## Как пользоваться
Выберите поле в меню сортировки — карточки перестроятся сразу. По дате добавления удобно ловить свежий импорт; по размеру или разрешению — искать тяжёлые файлы или материал для печати; по длительности — отбирать короткие референсы-анимации среди видео.
На **«Библиотеке»** пункт **«Перемешать»** выстраивает случайный порядок. Полезно, когда хотите заново взглянуть на старую базу «со свежей головой» или показать коллеге подборку без привязки к дате. Чтобы вернуть предсказуемый порядок, снова выберите, например, сортировку по дате.
Через **Меню → «Размер сетки»** переключайте масштаб. **Маленькая** — максимум карточек на экране, обзор всей стены. **Большая** — крупные превью, меньше прокрутки, если нужно оценить композицию до открытия карточки. **Средняя** — компромисс на каждый день.
Сортировка и размер независимы: можно держать компактную сетку и сортировку по дате или крупные превью со случайным порядком после **«Перемешать»**.
## Советы
По умолчанию новые карточки часто оказываются **сверху** при сортировке по дате от новых к старым. Если после импорта «не видите» файлы — проверьте направление сортировки и не включён ли случайный порядок.
**«Фильтры»** и сортировка запоминаются в рамках сессии работы с разделом. После жёсткого отбора не забывайте сбрасывать фильтры — иначе можно решить, что в базе мало карточек, хотя они просто скрыты.
На больших мониторах при **Большой** сетке ARC может отдавать более чёткие миниатюры — заметно на экранах с высокой плотностью пикселей.
В **«Коллекциях»** сортировка применяется только к карточкам **текущей** коллекции — не путайте с порядком во всей **«Библиотеке»**.
