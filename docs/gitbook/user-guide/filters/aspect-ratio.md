## О чём эта категория
Фильтр **«Соотношение сторон»** отбирает карточки по **форме кадра** — горизонтальные, вертикальные, квадратные и панорамные. Это удобно, когда задача уже диктует формат: вертикальные референсы для мобильного макета, квадратные для аватара, широкие панорамы для фона или cinematic-кадры для широкоформатной композиции.
Категория смотрит на пропорции изображения или кадра видео, а не на метки и не на сюжет. Два портрета с разными ярлыками попадут в одну группу, если оба вертикальные.
## Как выбирать значения
Откройте **«Фильтры»** → **«Соотношение сторон»** — справа появится второе меню с вариантами. Можно отметить **одно или несколько** соотношений; внутри категории действует логика «или» — подойдёт любой из выбранных форматов. Вместе с другими активными категориями условия **складываются**: карточка должна подходить и по сторонам, и по остальным фильтрам.
Рядом с каждым пунктом, если в базе есть подходящие карточки, показывается **счётчик** — сколько файлов попадает в этот вариант **в текущем контексте** (вся библиотека, выбранная коллекция или мудборд). Пункты без подходящих карточек могут **не отображаться** — это нормально для пустых комбинаций.
## Типичные сценарии
Часто **«Соотношение сторон»** сочетают с **«Тип файла»** — например, только горизонтальные изображения без GIF. Или с **«Дата добавления»**, чтобы собрать свежие вертикальные референсы за последнюю неделю. Поиск по меткам в центре навбара при этом продолжает работать: сначала смысл ярлыками, затем форма фильтром.
На вкладках **«Библиотека»**, **«Коллекции»** и **«Мудборд»** фильтр действует на видимую сетку. Сбросить только эту категорию можно в её втором меню; полный сброс — через **«Очистить фильтры»** в главном списке.
