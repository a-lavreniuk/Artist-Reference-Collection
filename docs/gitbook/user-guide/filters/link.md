## О чём эта категория
Фильтр **«Ссылка»** отбирает карточки по тому, **указан ли URL** у референса — ссылка на источник, пин, статью, Behance или другой адрес, откуда взят материал. Это отдельное поле в свойствах карточки, не путать с описанием или метками.
Категория помогает найти референсы без источника для дозаполнения или, наоборот, отфильтровать только те, где ссылка уже сохранена.
## Режимы «есть» и «нет»
Во втором меню доступны **«Есть ссылка»** и **«Нет ссылки»**. Счётчики показывают, сколько карточек в каждой группе в текущем контексте — библиотека, коллекция или мудборд. Режим **«Нет ссылки»** удобен для регулярной проверки: что импортировали, но ещё не привязали к источнику.
## Поиск по ключевым словам в URL
При выборе **«Есть ссылка»** может появиться поле **ключевых слов**. Введите фрагменты адреса или домена через пробел — останутся карточки, где **все** слова встречаются в URL (логика «И»). Например, можно отобрать всё с **pinterest** или с конкретным доменом портфолио.
Фильтр не ищет по тексту описания — только по полю ссылки. Для смыслового поиска по заметкам используйте категорию **«Описание»** или режим **«Метки»** в навбаре.
## Сочетания и контекст
На вкладках **«Библиотека»**, **«Коллекции»** и **«Мудборд»** условие складывается с другими фильтрами. Вместе с **«Описание»** получается выборка «полностью оформленных» карточек. Вместе с **«Тип файла»** — например, только видео с сохранённым источником.
Сбросить категорию можно в её меню или через **«Очистить фильтры»** в главном списке, когда активен хотя бы один фильтр.
