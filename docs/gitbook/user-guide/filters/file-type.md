## О чём эта категория
Фильтр **«Тип файла»** отбирает карточки по **формату исходника** — конкретным расширениям изображений и видео в вашей библиотеке. Удобно, когда нужны только статичные кадры без анимации, только видеореференсы, или вы готовите выгрузку и хотите видеть, например, все PNG или все MP4 в текущей выборке.
В отличие от поиска по меткам, здесь нет смысловых ярлыков — только технический тип файла, который ARC определил при импорте.
## Как устроен список расширений
Во втором меню категории перечислены расширения, которые **реально встречаются** в текущем контексте — **JPG**, **PNG**, **WEBP**, **MP4**, **GIF** и другие. Список не фиксированный каталог «всех форматов мира»: если в коллекции нет MKV, пункта MKV не будет.
Можно отметить несколько расширений сразу — внутри категории это «или». Счётчик рядом с пунктом показывает, сколько карточек с этим расширением есть сейчас в сетке. Изображения и видео часто удобно отбирать группами, отмечая сразу несколько близких форматов.
## Типичные сценарии
Перед презентацией можно оставить только **изображения**, убрав видео и GIF. Перед архивацией тяжёлых исходников — сочетать с **«Вес файла»**. Вместе с **«Соотношение сторон»** получается точная техническая выборка: «только горизонтальные PNG».
Фильтр работает на вкладках **«Библиотека»**, **«Коллекции»** и **«Мудборд»** и сочетается с поиском в центре навбара. Сортировка по **типу файлов** рядом с иконкой фильтров меняет порядок карточек, но не заменяет этот фильтр — сортировка не скрывает лишние форматы.
