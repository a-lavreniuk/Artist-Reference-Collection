В ARC используется один шрифт Google Sans с несколькими стилями начертания в зависимости от цели и размера.
# Стили заголовков
Далее идёт таблица с настройками шрифтов для заголовков. Приложение использует систему, похожую на ту, что применяется в веб-разработке. В этой системе заголовки обозначаются буквами **H**, где **H1** — самый большой заголовок, который обычно используется для главных разделов, а **H4** — самый маленький, который часто применяется для карточек.
H1. Шрифт для заголовков названий разделов.
```
font-family: "Google Sans";
font-size: 34px;
font-style: normal;
font-weight: 500;
line-height: 34px;
letter-spacing: 0;
paragraph-spacing: 0;

```
H2. Шрифт для заголовков страниц разделов второй степени.
```
font-family: "Google Sans";
font-size: 28px;
font-style: normal;
font-weight: 500;
line-height: 28px;
letter-spacing: 0;
paragraph-spacing: 0;

```
H3. Шрифт для заголовков страниц разделов третьей степени.
```
font-family: "Google Sans";
font-size: 22px;
font-style: normal;
font-weight: 500;
line-height: 22px;
letter-spacing: 0;
paragraph-spacing: 0;

```
# Стили текстов
Далее идёт таблица с настройками шрифтов для наборного. Приложение использует систему, похожую на ту, что применяется в веб-разработке.
**L** — шрифт для выделенных и ключевых текстов.
```
font-family: "Google Sans";
font-size: 16px;
font-style: normal;
font-weight: 400;
line-height: 24px
letter-spacing: 8px;
paragraph-spacing: 0;

```
**M** — основной шрифт для текстов.
```
font-family: "Google Sans";
font-size: 14px;
font-style: normal;
font-weight: 400;
line-height: 22px;
letter-spacing: 6px;
paragraph-spacing: 0;

```
**S** — шрифт для подписей и меток.
```
font-family: "Google Sans";
font-size: 12px;
font-style: normal;
font-weight: 400;
line-height: 16px;
letter-spacing: 0;
paragraph-spacing: 0;

```
