Приложение выполнено в тёмной цветовой гамме, что влияет на отображение элементов относительно друг друга. Чем выше элемент в системе дизайна, тем он светлее. Самые «нижние» уровни остаются тёмными.
Например, галерея находится на самом «нижнем» уровне. Закреплённое меню с поисковой строкой расположено на «среднем» уровне. Модальное окно с результатами поиска находится «выше» всех остальных элементов.
Для этих уровней предусмотрены три категории:
Sunken самый нижний уровень. Подходит для фонов и основных разделов;
[https://www.figma.com/design/JD3pZdlV4Sz62creRMQsJV/ARC-2?node-id=106-4946&t=3xMCPrY1tq0hvsAT-11](https://www.figma.com/design/JD3pZdlV4Sz62creRMQsJV/ARC-2?node-id=106-4946&t=3xMCPrY1tq0hvsAT-11)
Default средний уровень. Используется для блоков и разделов над основным контентом. Идеально для закреплённого меню и детальных карточек;
[https://www.figma.com/design/JD3pZdlV4Sz62creRMQsJV/ARC-2?node-id=106-4975&t=3xMCPrY1tq0hvsAT-11](https://www.figma.com/design/JD3pZdlV4Sz62creRMQsJV/ARC-2?node-id=106-4975&t=3xMCPrY1tq0hvsAT-11)
Raised высокий уровень. Предназначен для элементов, которые должны перекрывать другие и быть заметными для пользователя. Чаще всего это модальные окна, алерты и тосты.
[https://www.figma.com/design/JD3pZdlV4Sz62creRMQsJV/ARC-2?node-id=106-5004&t=3xMCPrY1tq0hvsAT-11](https://www.figma.com/design/JD3pZdlV4Sz62creRMQsJV/ARC-2?node-id=106-5004&t=3xMCPrY1tq0hvsAT-11)
При переходе от низкого уровня к высокому яркость цветовой палитры увеличивается на один уровень. Например, белый шрифт на Sunken выполнен в токене gray/300. На уровне Default это будет gray/200, а на Raised — gray/100.
Эта система помогает чётко разделить элементы интерфейса и делает взаимодействие с приложением более удобным для пользователя.
