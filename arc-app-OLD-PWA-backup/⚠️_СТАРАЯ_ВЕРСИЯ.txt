      - Работает в браузере
      - File System Access API
      - Service Worker
      - Требует разрешения браузера
    
    Electron (arc-electron/):
      - Настольное приложение
      - Node.js fs (прямой доступ)
      - electron-updater
      - Полный доступ к системе
