# Шрифты для ARC

Эта папка предназначена для хранения файлов шрифтов приложения.

## Необходимые шрифты

### 1. YS Music Headline Bold
Фирменный шрифт для заголовков.

**Файлы:**
- `YSMusic-HeadlineBold.ttf`
- `YSMusic-HeadlineBold.woff`
- `YSMusic-HeadlineBold.woff2` (рекомендуется для производительности)

**Скачать:** [Яндекс Диск](https://disk.yandex.ru/d/LMIDI9_fQzREiA)

### 2. Geologica Variable
Основной шрифт для текста.

**Файлы:**
- `Geologica-VariableFont.ttf`
- `Geologica-VariableFont.woff2` (рекомендуется для производительности)

**Скачать:** [Google Fonts - Geologica](https://fonts.google.com/specimen/Geologica)

## Установка

1. Скачай файлы шрифтов по ссылкам выше
2. Помести их в эту папку (`public/fonts/`)
3. Перезапусти dev-сервер если он запущен

## Примечание

Шрифты уже подключены в `src/styles/typography.css`, просто добавь файлы в эту папку.
