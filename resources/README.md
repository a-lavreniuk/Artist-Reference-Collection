# Resources

Эта папка содержит ресурсы для сборки приложения:

## icon.ico

Иконка приложения для Windows. Должна быть в формате .ico и содержать следующие размеры:
- 16x16
- 32x32
- 48x48
- 64x64
- 128x128
- 256x256

Вы можете создать .ico файл из PNG используя онлайн-конвертер:
- https://icoconvert.com/
- https://convertico.com/

## Создание иконки

1. Создайте PNG изображение 512×512px
2. Конвертируйте в .ico с множественными размерами
3. Поместите как `icon.ico` в эту папку

Пока используется placeholder иконка из Electron по умолчанию.
