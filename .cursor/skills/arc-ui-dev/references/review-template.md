# Шаблон UI-ревью ARC

Использовать при запросе «проверь UI», «ревью», «a11y», «сверь с макетом».

## 1. Контекст

- Файлы / экран
- Figma node (если есть)
- Тип изменения: новый элемент / правка / рефакторинг

## 2. Чеклист ARC (обязательный)

### Компоненты и паттерны

- [ ] Использован существующий общий компонент, а не локальный дубликат
- [ ] Нет копирования CSS из другого компонента «как есть»
- [ ] Empty state / tooltip / menu — через системные компоненты
- [ ] Тексты empty state — из `emptyStates.ts`

### Визуал и токены

- [ ] Только токены `--s-*`, `--brand-*`, `--typo-*`, transitions системные
- [ ] Нет произвольных `#hex`, лишних `px`/`ms`
- [ ] Типографика и отступы согласованы с Figma / UI-Kit

### Состояния

- [ ] `hover` — где интерактивно
- [ ] `focus` / `focus-visible` — клавиатура
- [ ] `disabled` — визуально и functionally
- [ ] `error` — поля форм (`field-error`, `aria-invalid`)
- [ ] `loading` — если есть async

### Поведение

- [ ] Мышь: клик, hover, drag (если применимо)
- [ ] Клавиатура: Tab order, Enter, Escape в модалках/меню
- [ ] Tooltip: delay, viewport, `pointer-events: none`
- [ ] Context menu: backdrop, Escape, 250px
- [ ] Electron: выделение текста и ввод не сломаны

### Формы

- [ ] Нет inline-текста ошибки под полем (rule Input-validation)
- [ ] Исключения (сеть/API) — только если продуктово согласовано

## 3. Дополнительно (по запросу)

Web Interface Guidelines (Vercel) — только пункты **без** эквивалента в ARC rules:

- semantic HTML, aria-labels
- `prefers-reduced-motion`
- touch targets, `touch-action`
- URL/state (если веб-навигация в renderer)

Формат finding: `path/to/file.tsx:42 — краткое описание`

## 4. Формат вывода

```markdown
## Результат для пользователя
[Что видно в интерфейсе]

## Находки
### Критично (нужно исправить)
- ...

### Рекомендации
- ...

### Ок
- ...

## Сверка с макетом
| Аспект | Статус | Комментарий |
|--------|--------|-------------|
| Отступы | ✅/⚠️/❌ | |
| Типографика | | |
| Состояния | | |

## Проверка
1. ...
2. ...
```

Уровни: **Критично** — ломает ARC rules или a11y; **Рекомендации** — улучшения без блокера.
