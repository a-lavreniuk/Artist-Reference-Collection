# Примеры — arc-anytype-bugs

## A. Триаж (фрагмент CSV)

**Вход:** строка с ID `2460017712`, тип «Производительность», критичность «Серьёзно — мешает, но есть обход».

**Действие:** одна карточка `bagi`, имя «Галерея зависает на прелоадерах при открытии», статус **Задача**.

**Не объединять** с ID `2460015810` (идея про список) — разная суть.

## B. Подхват «В работе»

**Пользователь:** перенёс «Галерея зависает на прелоадерах…» в **В работе**, пишет «возьми баги».

**Агент:**

1. `API-search-space` → один объект `bagi` со статусом «В работе».
2. `API-get-object` → читает шаги и обход (деталка открывается).
3. Ищет в коде загрузку превью masonry / gallery feed.
4. Чинит, отчитывается.
5. После «ок» → `API-update-object`, status **Готово**, в теле — «Итог: …».

## C. Тестовый отчёт

**Вход:** «Тест» / «Тестовая форма», критичность «Идея».

**Действие:** карточка `[Тест] …`, статус **Идея**, в теле пометка «можно архивировать». **Не** переводить в работу без запроса.
