### 2. Цвета иконок

- [ ] Активная иконка черная

- [ ] Кнопки закрытия меток работают

### 5. Специальные иконки
- [ ] `arrow-left` работает (встроенный SVG)

## Известные проблемы

### Отсутствующие файлы
Некоторые иконки используют встроенные SVG, так как файлов нет:
- `arrow-left-*-border.svg`
- `search-*-border.svg`

Если нужно добавить fill варианты или другие размеры - экспортируйте из Figma.

### Особые названия файлов
- `trash-16-border` → `trash--16-border.svg` (двойной дефис)
- `trash-*-fill` → `trash-3-*-fill.svg`
- `grid-*-fill` → `grid-2-*-fill.svg`

## Как проверить цвета в DevTools

1. Откройте DevTools (F12)
2. Выберите элемент с иконкой
3. В Computed посмотрите значение `color`
4. SVG должны использовать `currentColor` для stroke/fill
5. Проверьте что цвет меняется при hover/active

## Если иконка не загружается

1. Проверьте консоль на ошибки
2. Убедитесь что файл существует в `renderer/src/assets/icons/`
3. Проверьте формат названия: `[name]-[size]-[style].svg`
4. Проверьте что в SVG используется `stroke="currentColor"` или `fill="currentColor"`
