# ESLint Configuration

Конфигурационный файл ESLint для проверки качества кода.

Файл `.eslintrc.json` содержит настройки линтера для TypeScript и JavaScript кода проекта.

## Основные правила

- Проверка неиспользуемых переменных (игнорируются переменные, начинающиеся с `_`)
- Предупреждения при использовании типа `any`
- Ограничение использования `console` (разрешены только `console.warn` и `console.error`)

## Игнорируемые паттерны

- `dist/` — скомпилированные файлы
- `node_modules/` — зависимости
- `*.config.ts` — конфигурационные файлы
